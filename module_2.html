What is CSS?

CSS: Cascading Style Sheet. It adds make-up on your html. You can customize your html doument using CSS properties.

Types of CSS

There are 3 types of CSS.

  1. Inline CSS
  2. Internal of Embaded CSS
  3. External CSS

Inline CSS

Example: 

<p style="color:rgb(33, 135, 175);">This is inline css</p>

Output:

This is inline css

Internal or Embaded CSS

Used in head section inside <style> tag

Example: 

<style>
        p{
            color:"blue";
        }
</style>
